# EJS Template Engine in Express.js

## 📖 About

This folder demonstrates the implementation of the **EJS (Embedded JavaScript) Template Engine** in Express.js. EJS enables the creation of dynamic web pages by embedding JavaScript directly into HTML templates, making it easy to render dynamic content from the server.

---

## 🚀 Concepts Covered

- Configuring EJS as the template engine
- Rendering dynamic views using `res.render()`
- Passing data from the server to EJS templates
- Using EJS expressions and variables
- Creating reusable and dynamic web pages

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- EJS

## 📌 EJS Template Engine

EJS (Embedded JavaScript) is a template engine for Express.js that allows JavaScript code to be embedded within HTML. It enables developers to generate dynamic web pages by rendering server-side data directly into templates.

---

## 🎯 Learning Outcome

Through this concept, I learned how to configure EJS in an Express.js application, render dynamic views, pass data from the server to templates, and build dynamic web pages using server-side rendering.
