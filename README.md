# MVC Architecture using Express.js

## 📖 About

This project demonstrates the implementation of the **Model-View-Controller (MVC)** architecture using Express.js and MongoDB.

The application allows users to submit their **Name** and **Email**, stores the data in MongoDB using Mongoose, and displays all saved records on the web page.

---

## 🚀 Features

- User registration with Name and Email
- Store data in MongoDB
- Display all stored users
- MVC folder structure
- Dynamic web pages using EJS

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS

## 📌 MVC Architecture

### Model
Handles the application's data and interacts with the MongoDB database using Mongoose.

### View
Responsible for rendering the user interface using EJS templates.

### Controller
Contains the application logic and manages requests between the Model and the View.
